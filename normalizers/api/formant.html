vlnm.normalizers.formant¶

Formant intrinsic normalizers¶

The following normalizers are implemented in the vlnm.normalizers.formant module:

  • BarkNormalizer

  • ErbNormalizer

  • LogNormalizer

  • Log10Normalizer

  • MelNormalizer

class BarkNormalizer(formants=None, transform=None, rename=None, groupby=None, **kwargs)[source]¶

Normalise formants using the Bark scale Traunmüller (1990).

\[F_i^* = 26.81 \ln\left( 1 + \frac{F_i}{F_i + 1960} \right) - 0.53\]

To use this normalizer in the vlnm.normalize() function, use method='bark'.

Parameters

  • formants (list of str)

    The DataFrame columns containing the formant data. If omitted, any columns matching 'f0', 'f1', 'f2', 'f3', 'f4', or 'f5' will be used.

  • transform (callable(ndarray) returns ndarray)

    Replace the function that transforms formants from the Hz scale to the Bark scale. The function should take numpy array-compatible data structure (e.g., numpy.ndarray, pandas.DataFrame, etc.) and return the transformed data.

Other Parameters

  • rename

    Rename output columns. See renaming columns for details.

  • groupby

    One or more columns over which to group the data before normalization. See grouping data for details.

  • **kwargs

    Optional keyword arguments passed to the parent constructor.

Example

from vlnm import pb1952, BarkNormalizer

df = pb1952(['speaker', 'vowel', 'f1', 'f2'])
norm_df = BarkNormalizer(rename='{}*').normalize(df)
norm_df.head()
speaker vowel f1 f2 f1* f2*
0 1 iy 240 2280 2 .39473 13 .88670
1 1 iy 280 2400 2 .82125 14 .22780
2 1 ih 390 2030 3 .91932 13 .11018
3 1 ih 310 1980 3 .13128 12 .94305
