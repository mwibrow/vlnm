vlnm.normalizers.projection¶

Projection normalizers¶

This module contains normalizers which can be used to perform supervised or unsupervised dimensionality reduction on vowel formant data, for example, projecting \(F_0\), \(F_1\), \(F_2\), \(F_3\) data (i.e., four-dimensional data) onto two dimensions.

The following normalizers are implemented in the vlnm.normalizers.projection module:

  • FactorAnalysisNormalizer

  • FastICANormalizer

  • LDANormalizer

  • NMFNormalizer

  • PCANormalizer

class FactorAnalysisNormalizer(columns=None, n_components=2, rename=None, groupby=None, **kwargs)[source]¶

Normalize data using Factor Analayis (FA).

To use this normalizer in the vlnm.normalize() function, use method='factor-analysis'.

Parameters

  • columns (list of str)

    The columns of the DataFrame which contain the features to use in FA. This does not have to be formant data, but must be numeric.

  • n_components (int)

    The required number of components. Should be equal to or less than the number of columns specified.

Other Parameters

  • rename

    Rename output columns. See renaming columns for details.

  • groupby

    One or more columns over which to group the data before normalization. See grouping data for details.

  • **kwargs

    All other paremeters are passed to the constructor of the sklearn.decomposition.FactorAnalysis class.

Examples

from vlnm import pb1952, FactorAnalysisNormalizer

df = pb1952(['speaker', 'vowel', 'f0', 'f1', 'f2', 'f3'])
norm = FactorAnalysisNormalizer(
    columns=['f0', 'f1', 'f2', 'f3'],
    n_components=2,
    rename='{}*')
norm_df = norm.normalize(df)

class LDANormalizer(vowel='vowel', columns=None, n_components=2, rename=None, groupby=None, **kwargs)[source]¶

Normalize data using Linear Discriminant Analysis (LDA).

See Wang & van Heuven (2006) for an example of applying LDA to phonetic data.

To use this normalizer in the vlnm.normalize() function, use method='lda'.

Parameters

  • vowel (str)

    The DataFrame column which contains the vowel labels. If not given, defaults to 'vowel'.

  • columns (list of str)

    The columns of the DataFrame which contain the features to use in LDA. This does not have to be formant data, but must be numeric.

  • n_components (int)

    The required number of components. Should be equal to or less than the number of columns specified.

Other Parameters

  • rename

    Rename output columns. See renaming columns for details.

  • groupby

    One or more columns over which to group the data before normalization. See grouping data for details.

  • **kwargs

    All other paremeters are passed to the constructor of the sklearn.discriminant_analysis.LinearDiscriminantAnalysis class.
