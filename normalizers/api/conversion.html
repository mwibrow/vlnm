vlnm.conversion¶

Conversion¶

This module contains miscellaeneous functions for converting from Hz to other scales.

hz_to_bark(frq, method='traunmuller')[source]¶

Convert from Hz to Bark scale.

Parameters

  • frq (ndarray)

    The frequency data to convert.

  • method (str)

    The conversion method used, from the list below. If not given, defaults to 'traumuller'.

    • 'syrdal' conversion from Syrdal & Gopal (1986):

      \[F^\prime = 13 \arctan\left( 0.76\frac{F_c}{1000} \right) + 3.5 \arctan \left( \frac{F_c}{7500} \right)^2\]

      where

      \[\begin{split}F_c = \begin{cases} 150 & \mbox{when } F < 150 \\ F - 0.2(F - 150) & \mbox{when } 150 \leq F < 200 \\ F - 0.2(250 - F) & \mbox{when } 200\leq F < 250 \\ F & \mbox{otherwise} \end{cases}\end{split}\]

    • 'traumuller' conversion from Traunmüller (1990):

    \[F^\prime = 26.81 \left(\frac{F}{F + 1960}\right)\]

    • 'volk' conversion from Völk (2015):

    \[F^\prime = 32.12 \left( 1 - \left(1 + \left(\frac{F}{873.47}\right)^{1.18} \right)^{-0.4} \right)\]

    • 'zwicker' conversion from Zwicker & Terhardt (1980):

    \[F^\prime = 13 \arctan\left( 0.76\frac{F}{1000} \right) + 3.5 \arctan \left( \frac{F}{7500} \right)^2\]
Return type

ndarray

Returns

The converted data.

hz_to_erb(frq)[source]¶

Convert Hz to approximate ERB scale.

\[F^\prime = 21.4\log\left(1 + 0.00437F\right)\]

hz_to_mel(frq)[source]¶

Convert from Hz to mel scale.

The formula used here is the ‘natural-log’ equivalent of the formula given in O'Shaughnessy (1987, p.150):

\[F^\prime = 1127\log\left(1 + \frac{F}{700}\right)\]
