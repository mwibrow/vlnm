vlnm.registration¶

The vlnm.registration module implements helper functions for documenting normalizer classes and registering them for use with the normalize() function.

classify(vowel=None, formant=None, speaker=None)[source]¶

Decorator for classifying a normalizer class.

This adds the classify attribute to the class consisting of a dictionary containing the keys and values passed to the decorator.

This is used solely for documenting normalizers.

Parameters

  • vowel (str)

    One of intrinsic or extrinsic.

  • formant (str)

    One of intrinsic or extrinsic

  • speaker (str)

    One of intrinsic or extrinsic

Return type

callable()

Returns

A decorator function.

Example

@classify(vowel='extrinsic', formant='intrinsic', speaker='intrinsic')
class WattFabriciusNormalizer(CentroidNormalizer):
    # Rest of definition.

register(name)[source]¶

Decorator for registering a normalizer class.

Parameters

  • name (str)

The name to use for the class. Internally, the register_normalizer() function is called.

Return type

callable()

Returns

A decorator function.

Example

@register('custom-normalizer')
class CustomNormalizer(Normalizer):
    # Custom normalizer definition.
register_normalizer(klass, name, alias=None, index=None)[source]¶

Register a normalizer to be used with the normalize() function.

Parameters

  • klass (Normalizer) (Normalizer)

    A normalizer class.

  • name (str)

    Name that can be used to refer to access the class in the get_normalizer() function.

  • index (dict)

    A dictionary in which the normalizer class will be registered. If omitted, the global index will be used.

Example

class CustomNormalizer(Normalizer):
    # Custom normalizer definition.

register_normalizer(CustomNormalizer, 'custom-normalizer')
