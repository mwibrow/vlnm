vlnm.normalizers.base¶

Vowel normalizer base classes¶

The base module contains the base classes for vowel normalizers as well as the DefaultNormalizer which returns data unnormalized so can be used as a control when comparing normalization methods.

The following normalizers are implemented in the vlnm.normalizers.base module:

  • ChainNormalizer

  • DefaultNormalizer

class ChainNormalizer(normalizers=None)[source]¶

Run multiple normalizers in sequence.

To use this normalizer in the vlnm.normalize() function, use method='chain'.

Parameters

  • normalizers (list of str | list of Normalizer)

A list of normalizers.

Examples

from vlnm import (
    pb1952,
    ChainNormalizer, BarkNormalizer, LobanovNormalizer)

df = pb1952(['speaker', 'vowel', 'f1', 'f2'])
normalizers = [
    BarkNormalizer(rename='{}*'),
    LobanovNormalizer(formants=['f1*', 'f2*'])
]
norm_df = ChainNormalizer(normalizers).normalize(df)
norm_df.head()

0 1 iy 240 2280 -1 .52586 1 .58003
1 1 iy 280 2400 -1 .24016 1 .74941
2 1 ih 390 2030 -0 .50463 1 .19441
3 1 ih 310 1980 -1 .03249 1 .11142
4 1 eh 490 1870 0 .10672 0 .92120

class DefaultNormalizer(formants=None, rename=None, **kwargs)[source]¶

‘Default’ normalizer which returns formant data unaltered.

To use this normalizer in the vlnm.normalize() function, use method='default'.

Parameters

  • formants (list of str)

The DataFrame columns containing the formant data. If omitted, any columns matching 'f0', 'f1', 'f2', 'f3', 'f4', or 'f5' will be used.

Other Parameters

  • rename

    Rename output columns. See renaming columns for details.

  • **kwargs

    Optional keyword arguments passed to the parent constructor.

Examples

from vlnm import pb1952, DefaultNormalizer

df = pb1952(['speaker', 'vowel', 'f1', 'f2'])
norm_df = DefaultNormalizer(rename='{}*').normalize(df)
norm_df.head()
speaker vowel f1 f2 f1* f2*
0 1 iy 240 2280 240 2280
1 1 iy 280 2400 280 2400
2 1 ih 390 2030 390 2030
3 1 ih 310 1980 310 1980
4 1 eh 490 1870 490 1870

class FormantGenericNormalizer(formants=None, **kwargs)[source]¶

Base class for normalizers which require general list of formants.

The FormantGenericNormalizer should be used as the base class for normalizers whose implementation does not need to distinguish between specific formants.

This class can only be instantiated by child classes.
