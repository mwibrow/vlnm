class GerstmanNormalizer(formants=None, speaker='speaker', rename=None, groupby=None, **kwargs)[source]¶

Normalize formants according to Gerstman (1968).

Formants are normalized by subtracting the speaker’s minimum value for the formant and then divided by the speaker’s range for the formant. These value are then multipled by \(999\).

\[F_i^* = 999 \frac{F_i - \min{F_i}}{\max{F_i}}\]

To use this normalizer in the vlnm.normalize() function, use method='gerstman'.

Parameters

  • formants (list of str)

    The DataFrame columns containing the formant data. If omitted, any columns matching 'f0', 'f1', 'f2', 'f3', 'f4', or 'f5' will be used.

  • speaker (str)

    The DataFrame column which contains the speaker labels. If not given, defaults to 'speaker'.

Other Parameters

  • rename

    Rename output columns. See renaming columns for details.

  • groupby

    One or more columns over which to group the data before normalization. See grouping data for details.

  • **kwargs

    Optional keyword arguments passed to the parent constructor.

Example

from vlnm import pb1952, GerstmanNormalizer

df = pb1952(['speaker', 'vowel', 'f1', 'f2'])
norm_df = GerstmanNormalizer(rename='{}*').normalize(df)

1 1 iy 280 2400 71 .35714 999 .00000
2 1 ih 390 2030 267 .58929 747 .55102

class LCENormalizer(speaker='speaker', formants=None, rename=None, groupby=None, **kwargs)[source]¶

Normalize by dividing formants by their mamximum value for a speaker.

Formants are normalized by “linear compression or expansion” (Lobanov, 1971 p.607), that is by dividing each formant for a given speaker by it’s maximum value for that speaker:

\[F_i^* = \frac{F_i}{\max{F_i}}\]

To use this normalizer in the vlnm.normalize() function, use method='lce'.

class LobanovNormalizer(speaker='speaker', formants=None, rename=None, groupby=None, **kwargs)[source]¶

Normalize formants according to Lobanov (1971):

This uses the following formula:

\[F_i^* = \frac{F_i - \mu_{F_i}}{\sigma_{F_i}}\]

Where \(\mu_{F_i}\) is the mean of the formant \(F_i\) and \(\sigma_{F_i}\) is the “rms deviation” (Lobanov, 1971, p.606) of formant \(F_i\) from its mean for the speaker’s vowels. As noted by Thomas & Kendall (2007) this is equivalent to the standard deviation when the mean is zero, and for efficiency this is how the normalizer is implemented.

To use this normalizer in the vlnm.normalize() function, use method='lobanov'.

Parameters

  • formants (list of str)

    The DataFrame columns containing the formant data. If omitted, any columns matching 'f0', 'f1', 'f2', 'f3', 'f4', or 'f5' will be used.

  • speaker (str)

    The DataFrame column which contains the speaker labels. If not given, defaults to 'speaker'.

Other Parameters

  • rename

    Rename output columns. See renaming columns for details.

  • groupby

    One or more columns over which to group the data before normalization. See grouping data for details.

  • **kwargs

    Optional keyword arguments passed to the parent constructor.

Example

from vlnm import pb1952, LobanovNormalizer

class NearyGMNormalizer(formants=None, speaker='speaker', exp=False, rename=None, groupby=None, **kwargs)[source]¶

Normalize by subtracting the speaker’s mean log-transformed formants.

The Neary ‘Grand Mean’ normalizer log-transforms each formant and then substracts the mean log-transform of all formants under-considraton for a given speaker. This is usually \(F_1\) and \(F_2\) (Flynn & Foulkes, 2011) or \(F_1\), \(F_2\), and \(F_3\) (Clopper, 2009):

\[F_i^* = T\left( \log\left(F_i\right) - \frac{1}{n} \sum_{j=1}^{n}\mu_{\log\left(F_j\right)} \right)\]

Where \(T(x)=x\) or \(T(x)=\exp(x)\), and n is the highest formant index.

To use this normalizer in the vlnm.normalize() function, use method='nearygm'.

Parameters

  • formants (list of str)

    The DataFrame columns containing the formant data. If omitted, any columns matching 'f0', 'f1', 'f2', 'f3', 'f4', or 'f5' will be used.

  • speaker (str)

    The DataFrame column which contains the speaker labels. If not given, defaults to 'speaker'.

  • exp (bool)

    If True transform the normalized formants using the exponential function with base \(e\).

Other Parameters

  • rename

    Rename output columns. See renaming columns for details.

  • groupby

    One or more columns over which to group the data before normalization. See grouping data for details.

  • **kwargs

    Optional keyword arguments passed to the parent constructor.

Examples
