VLNM module¶

list_normalizers(sort=True, module=None, index=None)[source]¶

Return a list of available normalizers.

Parameters

  • sort (bool)

    Whether to sort the list by alphabetical order

  • module (str)

    List normalizers in the specified module. If omitted, all available normalizers will be listed.

  • index (dict)

    The register in which the normalizer was registered. If omitted, the global register will be used.

Return type

list of str

Returns

A list containing the names of the available normalizers.

Example

from vlnm import list_normalizers
list_normalizers()

normalize(data, file_out=None, method='default', sep=', ', **kwargs)[source]¶

Normalize vowel data.

Parameters

  • data (str | TextIOWrapper | DataFrame)

    A path to a CSV file containing the data, a file handle to an open file containing the data, or a Pandas DataFrame containing the data.

  • file_out (str | TextIOWrapper)

    An optional a file path or a file handle to which the output data will be saved (using DataFrame.to_csv).

  • method (str)

    The name of a normalization method. Method names can be found using the list_normalizers() function.

  • sep (str)

    The column separator in a file.

  • **kwargs

    Other keyword arguments passed on to the normalizer class.

Return type

DataFrame

Returns

If file_out is not specified, a Pandas DataFrame containing the normalized data.
