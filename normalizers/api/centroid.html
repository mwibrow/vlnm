vlnm.normalizers.centroid¶

Centroid normalizers¶

Centroid normalizers are speaker intrinsic normalizers which calculate the centroid (i.e., geometric center) of a speaker’s vowel space and use this to normalize the formant data by divided the formants for each vowel by the correspoinding formant of the centroid.

The following normalizers are implemented in the vlnm.normalizers.centroid module:

  • BighamNormalizer

  • CentroidNormalizer

  • ConvexHullNormalizer

  • SchwaNormalizer

  • WattFabricius1Normalizer

  • WattFabricius2Normalizer

  • WattFabricius3Normalizer

class BighamNormalizer(f1='f1', f2='f2', speaker='speaker', vowel='vowel', points=None, rename=None, groupby=None, **kwargs)[source]¶

Centroid normalizer using the centroid calculated according to Bigham (2008).

Bigham (2008) adapts Watt & Fabricius (2002) to calculate a barycentric centroid from a trapezoid constructed using the vowels [i\(^\prime\)], [u\(^\prime\)] [ɑ\(^\prime\)], and [æ\(^\prime\)] derived as shown below:

Construction of derived vowels¶

vowel

F1

F2

[i\(^\prime\)]

\(F_1^{[ɪ]}\)

\(F_2^{[i]}\)

[u\(^\prime\)]

\(F_1^{[i]}\)

\(F_2^{[u]}\)

[ɑ\(^\prime\)]

\(\frac{1}{2}(F_1^{[ɑ]} + F_1^{[ɔ]})\)

\(\frac{1}{2}(F_2^{[ɑ]} + F_2^{[ɔ]})\)

[æ\(^\prime\)]

\(F_1^{[æ]}\)

\(F_2^{[æ]}\)

Then the formants are normalized as follows:

\[F_i^* = \frac{F_i}{S(F_i)}\]

Where:

\[S(F_i) = \frac{1}{4} \left( F_i^{[i^\prime]} + F_i^{[u^\prime]} + F_i^{[ɑ^\prime]} + F_i^{[æ^\prime]} \right)\]

To use this normalizer in the vlnm.normalize() function, use method='bigham'.

Parameters

  • f1 (str)

    DataFrame columns containing \(F_1\) data. If omitted, defaults to 'f1'.

  • f2 (str)

    DataFrame columns containing \(F_2\) data. If omitted, defaults to 'f2'.

  • speaker (str)

    The DataFrame column which contains the speaker labels. If not given, defaults to 'speaker'.

  • vowel (str)

    The DataFrame column which contains the vowel labels. If not given, defaults to 'vowel'.

  • points (dict)

    A dictionary specifying labels for the required vowels to construct the centroid (shown in the table above). The keys for the dictionary should be from the lexical set keywords Wells (1982): 'kit', 'goose', 'fleece', 'start', 'thought', 'trap', and all keys need to be specified. If omitted, the normalizer will assume that the vowels are already labeled according to the lexical set keywords.

Other Parameters

  • rename

    Rename output columns. See renaming columns for details.

  • groupby

    One or more columns over which to group the data before normalization. See grouping data for details.

  • **kwargs

    Optional keyword arguments passed to the parent constructor.

Examples

from vlnm import pb1952, BighamNormalizer

df = pb1952(['speaker', 'vowel', 'f1', 'f2'])
points = dict(
    kit='ih',
    goose='uw',
    fleece='iy',
    start='aa',
    thought='ao',
    trap='ae')
norm = BighamNormalizer(points=points, rename='{}*')
norm_df = norm.normalize(df)

class CentroidNormalizer(formants=None, speaker='speaker', vowel='vowel', points=None, rename=None, groupby=None, **kwargs)[source]¶

Normalize using the geometric center of the speakers entire vowel space.

For a given speaker, the normalized formants are calculated as follows:

\[F_i^* = \frac{F_i}{S_i}\]

where

\[S_i = \frac{1}{|J|}\sum_{j\in J}\mu_{F_{ij}}\]

Where \(J\) is the the set of vowels which form the convex hull of the vowel space, and \(\mu_{F_{ij}}\) is the mean of formant \(i\) for vowel \(j\).

To use this normalizer in the vlnm.normalize() function, use method='centroid'.

Parameters

  • formants (list of str)

    The DataFrame columns containing the formant data. If omitted, any columns matching 'f0', 'f1', 'f2', 'f3', 'f4', or 'f5' will be used.

  • speaker (str)

    The DataFrame column which contains the speaker labels. If not given, defaults to 'speaker'.

  • vowel (str)

    The DataFrame column which contains the vowel labels. If not given, defaults to 'vowel'.

  • points (list of str)

    List of vowel labels corresponding to each ‘corner’ of the speakers vowel space. If omitted all vowels will be used.

Other Parameters

  • rename

    Rename output columns. See renaming columns for details.

  • groupby

    One or more columns over which to group the data before normalization. See grouping data for details.

  • **kwargs

    Optional keyword arguments passed to the parent constructor.

class ConvexHullNormalizer(formants=None, speaker='speaker', vowel='vowel', where='mean', rename=None, groupby=None, **kwargs)[source]¶

Normalize using the geometric center of the convex hull enclosing the speakers vowel space.

The convex hull normalizer establishes the speaker’s vowel space by calulating the convex hull (e.g., Graham & Yao, 1983) from the ‘central’ formants (i.e., mean or median) for each of the speaker’s vowels, and uses the barycenter of the points that make-up the hull to normalize the formant data.

For a given speaker, the normalized formants are calculated as follows:

\[F_i^* = \frac{F_i}{S_i}\]

where

\[S_i = \frac{1}{|J|}\sum_{j\in J}||F_{ij}||\]

Where \(J\) is the the set of vowels which form the convex hull of the vowel space, \(F_{ij}\) is formant \(i\) of vowel \(j\), and \(||F_{ij}||\) indicates the central tendency of \(F_{ij}\).

To use this normalizer in the vlnm.normalize() function, use method='convex-hull'.

Parameters

  • formants

    The DataFrame columns containing the formant data. If omitted, any columns matching 'f0', 'f1', 'f2', 'f3', 'f4', or 'f5' will be used.

  • speaker

    The DataFrame column which contains the speaker labels. If not given, defaults to 'speaker'.

  • vowel

    The DataFrame column which contains the vowel labels. If not given, defaults to 'vowel'.

  • where

    Central tendency measure to use when calculating the points of the convex hull. One of 'mean' (the default) or 'median'.

Other Parameters

  • rename

    Rename output columns. See renaming columns for details.

  • groupby

    One or more columns over which to group the data before normalization. See grouping data for details.

  • **kwargs

    Optional keyword arguments passed to the parent constructor.

from vlnm import pb1952, ConvexHullNormalizer

class WattFabricius1Normalizer(f1='f1', f2='f2', speaker='speaker', vowel='vowel', fleece='fleece', trap='trap', points=None, rename=None, groupby=None, **kwargs)[source]¶

Normalize vowels according to Watt & Fabricius (2002).

Formant data is normalized by by dividing the component formants for a vowel by a the components of a centroid calculated from the \(F_1\) and \(F_2\) data for the fleece, trap and goose vowels, with the formants for the goose vowel derived from the other vowels by setting \(F_1\) and \(F_2\) to the \(F_1\) of the fleece vowel.

\[F_i^* = \frac{F_i}{S(F_i)}\]

Where:

\[S(F_j) = \frac{1}{3}\left (F_j^{[i]} + F_j^{[a]} + F_j^{[u^\prime]} \right)\]

and

\[F_1^{[u^\prime]} = F_2^{[u^\prime]} = F_1^{[i]}\]

with \([i]\), \([a]\), and \([u^\prime]\) indicating the fleece, trap and (derived) goose vowels, respectively.

To use this normalizer in the vlnm.normalize() function, use method='wattfab1'.

Parameters

  • f1 (str | list of str)

    DataFrame columns containing \(F_1\) data. If omitted, defaults to 'f1'.

  • f2 (str | list of str)

    DataFrame columns containing \(F_2\) data. If omitted, defaults to 'f2'.

  • speaker (str)

    The DataFrame column which contains the speaker labels. If not given, defaults to 'speaker'.

  • vowel (str)

    The DataFrame column which contains the vowel labels. If not given, defaults to 'vowel'.

  • fleece (str)

    Vowel label corresponding to the fleece vowel. If omitted, defaults to 'fleece'.

  • trap (str)

    Vowel label corresponding to the trap vowel. If omitted, defaults to 'trap'.

  • points (dict)

    Alternative method for specifying the fleece and trap labels, consisting of a dictionary with the keys 'fleece' and 'trap', whose values correspond to their respective labels.

Other Parameters

  • rename

    Rename output columns. See renaming columns for details.

  • groupby

    One or more columns over which to group the data before normalization. See grouping data for details.

  • **kwargs

    Optional keyword arguments passed to the parent constructor.

Second centroid normalizer described in Watt & Fabricius (2002).

\[F_i^* = \frac{F_i}{S(F_i)}\]

Where:

\[\begin{split}S(F_j) = \begin{cases} \frac{1}{2}\left(F_j^{[i]} + F_j^{[u^\prime]}\right) & \text{when } j = 2 \\ \frac{1}{3}\left(F_j^{[i]} + F_j^{[a]} + F_j^{[u^\prime]}\right) & \text{otherwise} \end{cases}\end{split}\]

and

\[F_1^{[u^\prime]} = F_2^{[u^\prime]} = F_1^{[i]}\]

class WattFabricius3Normalizer(f1='f1', f2='f2', speaker='speaker', vowel='vowel', fleece='fleece', trap='trap', points=None, rename=None, groupby=None, **kwargs)[source]¶

Third centroid normalizer described in Watt & Fabricius (2002).

\[F_i^* = \frac{F_i}{S(F_i)}\]

Where:

\[\begin{split}S(F_j) = \begin{cases} \frac{1}{2}\left(F_j^{[i]} + F_j^{[u^\prime]}\right) & \text{when } j = 2 \\ \frac{1}{3}\left(F_j^{[i]} + F_j^{[a]} + F_j^{[u^\prime]}\right) & \text{otherwise} \end{cases}\end{split}\]

and

\[F_j^{[u^\prime]} = \underset{\rho}{\text{argmin}}\mbox{ }\mu_{F_k^{/\rho \in P/}}\]

where \(P\) is the set of point vowels.

To use this normalizer in the vlnm.normalize() function, use method='wattfab3'.
