vlnm.normalizers.gender¶

Gender-based normalizers¶

Normalizers which adjust calculations based on the speaker’s identified/identifying gender. Note, that as defined in the literature (e.g., Bladon et al., 1984; Nordström, 1977) the normalizers use a binary gender classification.

The following normalizers are implemented in the vlnm.normalizers.gender module:

  • BladenNormalizer

  • NordstromNormalizer

class BladenNormalizer(formants=None, gender='gender', female='F', male='M', rename=None, groupby=None, **kwargs)[source]¶

Normalize formant data according to Bladon et al. (1984).

For a given speaker, formants are normalized as follows:

\[F_{i}^* = 26.81 \left( 1 + \frac{F_i}{F_i + 1960} \right) - 0.53 - I(F_i)\]

Where \(I(F_i)\) is an indicator function returning 1 if \(F_i\) was produced by a speaker identified/identifying as female, and 0 otherwise.

To use this normalizer in the vlnm.normalize() function, use method='bladen'.

Parameters

  • formants (list of str)

    The DataFrame columns containing the formant data. If omitted, any columns matching 'f0', 'f1', 'f2', 'f3', 'f4', or 'f5' will be used.

  • gender (str)

    The DataFrame column containing the gender labels.

  • female (str)

    The label in the DataFrame indicating a speaker identified/identifying as female. If omitted, defaults to F.

  • male (str)

    The label in the DataFrame indicating a speaker identified/identifying as male. If omitted, defaults to M.

Other Parameters

  • rename

    Rename output columns. See renaming columns for details.

  • groupby

    One or more columns over which to group the data before normalization. See grouping data for details.

  • **kwargs

    Optional keyword arguments passed to the parent constructor.

Examples

from vlnm import pb1952, BladenNormalizer

df = pb1952(['speaker', 'sex', 'vowel', 'f1', 'f2'])
norm = BladenNormalizer(
    formants=['f1', 'f2'], gender='sex', female='f', rename='{}*')
norm_df = norm.normalize(df)
norm_df.head()

class NordstromNormalizer(f0=None, f1=None, f2=None, f3=None, gender='gender', female='F', male='M', rename=None, groupby=None, **kwargs)[source]¶

Normalize formant data according to Nordström (1977).

\[\begin{split}F_i^* = F_i \times \begin{cases} \frac{ \mu_{F_3}^{\small{male}} }{ \mu_{F_3}^{\small{female}} } & \mbox{if } I(F_i) = 1 \\ 1 & \mbox{otherwise} \end{cases}\end{split}\]

Where \(\mu_{F_3}\) is the mean \(F_3\) across all vowels where \(F_1\) is greater than 600Hz, and \(I(F_i)\) is an indicator function which returns 1 if \(F_i\) is from a speaker identified/identifying as female, and 0 otherwise.

To use this normalizer in the vlnm.normalize() function, use method='nordstrom'.

Parameters

  • f0, f1, f2, f3

  • gender (str)

    The DataFrame column containing the gender labels.

  • female (str)

    The label in the DataFrame indicating a speaker identified/identifying as female.

  • male (str)

    The label in the DataFrame indicating a speaker identified/identifying as male.

Other Parameters

  • rename

    Rename output columns. See renaming columns for details.

  • groupby

    One or more columns over which to group the data before normalization. See grouping data for details.

  • **kwargs

    Optional keyword arguments passed to the parent constructor.

Examples

from vlnm import pb1952, NordstromNormalizer

df = pb1952(['speaker', 'sex', 'vowel', 'f1', 'f2', 'f3'])
norm = NordstromNormalizer(gender='sex', female='f', rename='{}*')
norm_df = norm.normalize(df)
