Vowel Normalization¶

VLNM Provides a comprehensive range of normalizers (see section Normalizers below) that have been described in the literature (although remains agnostic regarding which may be considered the most suitable for a given collection of formant data).

To just get started using normalizers, jump to Getting started. More detailed descriptions of the implementation (including links to the source code) can be found in the API.

Normalizers¶

VLNM implements the normalizers shown below. Summmaries for each normalizer can be viewed by clicking on the name in the Normalizer column. The classification of normalizers according to whether the calculations they perform are intrinsic or extrinsic to certain properties (i.e., vowel, formant, and speaker) are taken for most normalizers from Flynn & Foulkes (2011).

The following sections provide brief summaries for each normalizer. Clicking on the Python class name for each normalizer will show the API documentation which includes details on how to specify parameters when using the normalizer.

bark

Normalise formants using the Bark scale Traunmüller (1990).

\[F_i^* = 26.81 \ln\left( 1 + \frac{F_i}{F_i + 1960} \right) - 0.53\]

The bark normalizer is implemented in the BarkNormalizer class.

barkdiff

Normalize formant data according to Syrdal & Gopal (1986).

Vowels are normalized by converting formants to the Bark scale, calculating the difference \(Z\) between consecutive formants:

\[F_{i}^* = B(F_i) - B(F_{i-1})\mbox{ for } 1 \leq i \leq 3\]

Where \(B\) is a function converting the \(i\mbox{th}\) formant measured in hertz to the Bark scale.

The barkdiff normalizer is implemented in the BarkDifferenceNormalizer class.

bigham

Centroid normalizer using the centroid calculated according to Bigham (2008).

Bigham (2008) adapts Watt & Fabricius (2002) to calculate a barycentric centroid from a trapezoid constructed using the vowels [i\(^\prime\)], [u\(^\prime\)] [ɑ\(^\prime\)], and [æ\(^\prime\)] derived as shown below:

Then the formants are normalized as follows:

\[F_i^* = \frac{F_i}{S(F_i)}\]

Where:

\[S(F_i) = \frac{1}{4} \left( F_i^{[i^\prime]} + F_i^{[u^\prime]} + F_i^{[ɑ^\prime]} + F_i^{[æ^\prime]} \right)\]

The bigham normalizer is implemented in the BighamNormalizer class.

bladen

Normalize formant data according to Bladon et al. (1984).

For a given speaker, formants are normalized as follows:

\[F_{i}^* = 26.81 \left( 1 + \frac{F_i}{F_i + 1960} \right) - 0.53 - I(F_i)\]

Where \(I(F_i)\) is an indicator function returning 1 if \(F_i\) was produced by a speaker identified/identifying as female, and 0 otherwise.

The bladen normalizer is implemented in the BladenNormalizer class.

centroid

Normalize using the geometric center of the speakers entire vowel space.

For a given speaker, the normalized formants are calculated as follows:

\[F_i^* = \frac{F_i}{S_i}\]

where

\[S_i = \frac{1}{|J|}\sum_{j\in J}\mu_{F_{ij}}\]

Where \(J\) is the the set of vowels which form the convex hull of the vowel space, and \(\mu_{F_{ij}}\) is the mean of formant \(i\) for vowel \(j\).

The centroid normalizer is implemented in the CentroidNormalizer class.

chain

Run multiple normalizers in sequence.

The chain normalizer is implemented in the ChainNormalizer class.

convex-hull

Normalize using the geometric center of the convex hull enclosing the speakers vowel space.

The convex hull normalizer establishes the speaker’s vowel space by calulating the convex hull (e.g., Graham & Yao, 1983) from the ‘central’ formants (i.e., mean or median) for each of the speaker’s vowels, and uses the barycenter of the points that make-up the hull to normalize the formant data.

For a given speaker, the normalized formants are calculated as follows:

\[F_i^* = \frac{F_i}{S_i}\]

where

\[S_i = \frac{1}{|J|}\sum_{j\in J}||F_{ij}||\]

Where \(J\) is the the set of vowels which form the convex hull of the vowel space, \(F_{ij}\) is formant \(i\) of vowel \(j\), and \(||F_{ij}||\) indicates the central tendency of \(F_{ij}\).

The convex-hull normalizer is implemented in the ConvexHullNormalizer class.

default

‘Default’ normalizer which returns formant data unaltered.

The default normalizer is implemented in the DefaultNormalizer class.

erb

Normalise formants using the ERB scale (Moore & Glasberg, 1996).

\[F_i^* = 21.4 \ln\left(1 + 0.00437{F_i}\right)\]

The erb normalizer is implemented in the ErbNormalizer class.

factor-analysis

Normalize data using Factor Analayis (FA).

The factor-analysis normalizer is implemented in the FactorAnalysisNormalizer class.

fast-ica

Normalize data using (fast) Independent Components Analysis (ICA).

The fast-ica normalizer is implemented in the FastICANormalizer class.

gerstman

Normalize formants according to Gerstman (1968).

Formants are normalized by subtracting the speaker’s minimum value for the formant and then divided by the speaker’s range for the formant. These value are then multipled by \(999\).

\[F_i^* = 999 \frac{F_i - \min{F_i}}{\max{F_i}}\]

The gerstman normalizer is implemented in the GerstmanNormalizer class.

ie-gmagm

Normalize formants according to Ananthapadmanabha & Ramakrishnan (2016, appendix B).

Formants for a given token are first normalized by dividing the values in Hz by the geometric mean of the first three formants. The normalized values are then ‘denormalized’ by multiplying by the geometric mean of the means for each vowel across all speakers. Equivalently, for a given token of vowel \(j\):

\[F^*_{ij} = F_{ij}\left( \prod_{i=1}^{3} \frac{\mu_{ij}}{F_{ij}} \right)^{\frac{1}{3}}\]

Where \(\mu_{ij}\) is the mean of \(F_i\) for vowel \(j\) for all tokens and speakers.

The ie-gmagm normalizer is implemented in the IEGMAGMNormalizer class.

ie-ht

Normalize (and potentially relabel) formants according in Ananthapadmanabha & Ramakrishnan (2016).

Let \(F^{\prime}_{ij}\) be the raw formant value in Hz for a particular token of vowel \(j\) divided by the gemometric mean of the first three formants for that token:

\[F^{\prime}_{ij} = \frac{F_{ij}} { \left( \prod_{i=1}^{3}F_{ij} \right)^{\frac{1}{3}} }\]

Let \(\mu_{ij}\) be the mean of \(F_i\) for vowel \(j\) for all tokens and speakers, and define \(F^{\prime\prime}_{ij}\) as follows:

\[F^{\prime\prime}_{ij} = F^{\prime}_{ij}\mu_{ij}\]

Then let \(\mu^{\prime\prime}_{ij}\) and \(\sigma^{\prime\prime}_{ij}\) be the mean and standard deviation, respectively, of \(F^{\prime\prime}_i\) for vowel \(j\) for all tokens and speakers. Then the normalized value can be calulated as follows:

\[F^*_{ij^*} = F^\prime_{ij} \mu_{ij^*}\]

Where \(j^*\) is the vowel from the set of vowels \(J\) such that:

\[j^* = \underset{j \in J}{\text{argmin}} \left( \sum_{i=1}^{2}\left( \frac{F^\prime_{ij}\mu_{ij} - \mu^{\prime\prime}_{ij}} {\sigma^{\prime\prime}_{ij}} \right)^2 \right)^{\frac{1}{2}}\]

The ie-ht normalizer is implemented in the IEHTNormalizer class.

lce

Normalize by dividing formants by their mamximum value for a speaker.

Formants are normalized by “linear compression or expansion” (Lobanov, 1971 p.607), that is by dividing each formant for a given speaker by it’s maximum value for that speaker:

\[F_i^* = \frac{F_i}{\max{F_i}}\]

The lce normalizer is implemented in the LCENormalizer class.

lda

Normalize data using Linear Discriminant Analysis (LDA).

See Wang & van Heuven (2006) for an example of applying LDA to phonetic data.

The lda normalizer is implemented in the LDANormalizer class.

lobanov

Normalize formants according to Lobanov (1971):

This uses the following formula:

\[F_i^* = \frac{F_i - \mu_{F_i}}{\sigma_{F_i}}\]

Where \(\mu_{F_i}\) is the mean of the formant \(F_i\) and \(\sigma_{F_i}\) is the “rms deviation” (Lobanov, 1971, p.606) of formant \(F_i\) from its mean for the speaker’s vowels. As noted by Thomas & Kendall (2007) this is equivalent to the standard deviation when the mean is zero, and for efficiency this is how the normalizer is implemented.

The lobanov normalizer is implemented in the LobanovNormalizer class.

log

Normalize using the natural logarithm of the formant values.

\[F_i^* = \log\left(F_i\right)\]

The log normalizer is implemented in the LogNormalizer class.

log10

Normalize using the base 10 logarithm of the formant values.

\[F_i^* = \log_{10}\left(F_i\right)\]

The log10 normalizer is implemented in the Log10Normalizer class.

mel

Normalise vowels using the Mel scale (see O'Shaughnessy, 1987, p.150).

\[F_i^* = 1127 \ln\left(1 + \frac{F_i}{700}\right)\]

The mel normalizer is implemented in the MelNormalizer class.

neary

Normalize by subtracting log-transformed formant means for each speaker.

\[F_i^* = T\left( \log\left(F_i\right) - \frac{1}{n} \sum_{j=1}^{n}\mu_{\log\left(F_j\right)} \right)\]

Where \(T(x)=x\) or \(T(x)=\exp(x)\), and n is the highest formant index.

The neary normalizer is implemented in the NearyNormalizer class.

neary-exp

NearyNormalizer with the exp parameter automatically set to True.

See Thomas & Kendall, 2007 for discussion.

The neary-exp normalizer is implemented in the NearyExpNormalizer class.

nearygm

Normalize by subtracting the speaker’s mean log-transformed formants.

The Neary ‘Grand Mean’ normalizer log-transforms each formant and then substracts the mean log-transform of all formants under-considraton for a given speaker. This is usually \(F_1\) and \(F_2\) (Flynn & Foulkes, 2011) or \(F_1\), \(F_2\), and \(F_3\) (Clopper, 2009):

\[F_i^* = T\left( \log\left(F_i\right) - \frac{1}{n} \sum_{j=1}^{n}\mu_{\log\left(F_j\right)} \right)\]

Where \(T(x)=x\) or \(T(x)=\exp(x)\), and n is the highest formant index.

The nearygm normalizer is implemented in the NearyGMNormalizer class.

nearygm-exp

NearyGMNormalizer with the exp parameter automatically set to True.

See Flynn & Foulkes (2011) and Thomas & Kendall (2007) for discussion.

The nearygm-exp normalizer is implemented in the NearyGMExpNormalizer class.

nmf

Normalize data using Non-negative Matrix Factorization (NMF).

The nmf normalizer is implemented in the NMFNormalizer class.

nordstrom

Normalize formant data according to Nordström (1977).

\[\begin{split}F_i^* = F_i \times \begin{cases} \frac{ \mu_{F_3}^{\small{male}} }{ \mu_{F_3}^{\small{female}} } & \mbox{if } I(F_i) = 1 \\ 1 & \mbox{otherwise} \end{cases}\end{split}\]

Where \(\mu_{F_3}\) is the mean \(F_3\) across all vowels where \(F_1\) is greater than 600Hz, and \(I(F_i)\) is an indicator function which returns 1 if \(F_i\) is from a speaker identified/identifying as female, and 0 otherwise.

The nordstrom normalizer is implemented in the NordstromNormalizer class.

pca

Normalize data using Principle Components Analysis (PCA).

See Jacobi et al. (2006) for an example of applying PCA to phonetic data.

The pca normalizer is implemented in the PCANormalizer class.

schwa

Centroid normalizer using formant data for [ə] as the centroid.

\[F_i^* = \frac{F_i}{F_{i}^{[ə]}} - 1\]

The schwa normalizer is implemented in the SchwaNormalizer class.

wattfab1

Normalize vowels according to Watt & Fabricius (2002).

Formant data is normalized by by dividing the component formants for a vowel by a the components of a centroid calculated from the \(F_1\) and \(F_2\) data for the fleece, trap and goose vowels, with the formants for the goose vowel derived from the other vowels by setting \(F_1\) and \(F_2\) to the \(F_1\) of the fleece vowel.

\[F_i^* = \frac{F_i}{S(F_i)}\]

Where:

\[S(F_j) = \frac{1}{3}\left (F_j^{[i]} + F_j^{[a]} + F_j^{[u^\prime]} \right)\]

and

\[F_1^{[u^\prime]} = F_2^{[u^\prime]} = F_1^{[i]}\]

with \([i]\), \([a]\), and \([u^\prime]\) indicating the fleece, trap and (derived) goose vowels, respectively.

The wattfab1 normalizer is implemented in the WattFabricius1Normalizer class.

wattfab2

Second centroid normalizer described in Watt & Fabricius (2002).

\[F_i^* = \frac{F_i}{S(F_i)}\]

Where:

\[\begin{split}S(F_j) = \begin{cases} \frac{1}{2}\left(F_j^{[i]} + F_j^{[u^\prime]}\right) & \text{when } j = 2 \\ \frac{1}{3}\left(F_j^{[i]} + F_j^{[a]} + F_j^{[u^\prime]}\right) & \text{otherwise} \end{cases}\end{split}\]

and

\[F_1^{[u^\prime]} = F_2^{[u^\prime]} = F_1^{[i]}\]

The wattfab2 normalizer is implemented in the WattFabricius2Normalizer class.

wattfab3

Third centroid normalizer described in Watt & Fabricius (2002).

\[F_i^* = \frac{F_i}{S(F_i)}\]

Where:

\[\begin{split}S(F_j) = \begin{cases} \frac{1}{2}\left(F_j^{[i]} + F_j^{[u^\prime]}\right) & \text{when } j = 2 \\ \frac{1}{3}\left(F_j^{[i]} + F_j^{[a]} + F_j^{[u^\prime]}\right) & \text{otherwise} \end{cases}\end{split}\]

and

\[F_j^{[u^\prime]} = \underset{\rho}{\text{argmin}}\mbox{ }\mu_{F_k^{/\rho \in P/}}\]

where \(P\) is the set of point vowels.

The wattfab3 normalizer is implemented in the WattFabricius3Normalizer class.
