vlnm.data¶

Datasets¶

VLNM distributes a small number of datasets containing formant data, which can be used to test or evaluate normalizers.

class Dataset(source, dtypes=None, **kwargs)[source]¶

Base class for datasets.

Parameters

  • source (str)

    Absolute file path to the dataset source. Must be readable by pd.read_csv().

  • dtypes (dict)

    Dictionary mapping column names on data types.

Other Parameters

  • **kwargs

Passed on to the pd.read_csv() function when data is loaded.

static enable_cache(enable=True)[source]¶

Enable caching for datasets.

By default data sets are loaded from disk. When the cache is enabled, data sets will be loaded in to memory, and subsequent calls to the load() method will return a copy of this data set.

Parameters

  • enable (bool)

If True enable and reset the cache.

load(columns=None, dtypes=None)[source]¶

Load a dataset from disk or the cache.

Parameters

  • columns (list of str)

    List of columns to subset the data. If omitted, all columns are returned.

  • dtypes (dict)

    Dictionary mapping column names on data types.

Returns

pd.DataFrame containing the data.

hm2005(columns=None, **kwargs)[source]¶

Data derived from Hawkins & Midgley (2005).

The data consists of formant measurements taken from 11 RP monopthongs in hVd contexts produced by twenty male speakers of RP divided into four age groups.

Parameters

  • columns (list of str)

    Specify which columns to return. If omitted all columns are returned.

  • **kwargs

    Passed on to vlnm.data.Dataset.load() method

Return type

DataFrame

Returns

A Dataframe containing the data

Dataset columns

  • group (pandas.Categorical of object)

    Group

  • age (pandas.Categorical of object)

    Age label of group

  • speaker (pandas.Categorical of np.int64)

    Speaker identifier

  • f1 - f2 (np.int64)

    Formant data in Hz

  • vowel (pandas.Categorical of object)

    Vowel labels as hVd words.

Examples

from vlnm.data import hm2005

hm2005(['speaker', 'age', 'vowel', 'f1', 'f2']).head()

1 S1-1 65+ hid 397 1877
2 S1-1 65+ head 436 1885
3 S1-1 65+ had 485 1545
4 S1-1 65+ hard 603 996
pb1952(columns=None, **kwargs)[source]¶

Return data derived from Peterson & Barney (1952).

The dataset was extracted from the source files for PRAAT (Boersma & Weenink, 2018) and the IPA symbols provided in those files replaced with their unicode equivalents.

Parameters

  • columns (list of str)

    Specify which columns to return. If omitted all columns are returned.

  • **kwargs

    Passed on to vlnm.data.Dataset.load() method

Return type

DataFrame

Returns

A Dataframe containing the data

Dataset columns

  • type (pandas.Categorical of object)

    Type of speaker (child, man or woman)

  • sex (pandas.Categorical of object)

    Reported gender of speaker

  • speaker (pandas.Categorical of np.int64)

    Speaker identifier

  • vowel (pandas.Categorical of object)

    Vowel labels

  • IPA (pandas.Categorical of object)

    IPA symbol

  • f0 (np.int64)

    Fundamental frequency in Hz.

  • f1 - f3 (np.int64)

    Formant data in Hz.
